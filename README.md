# パネル開けツール Web版

iPhone や Android のブラウザで使える静的Webアプリ版です。

## 使い方

`index.html` をブラウザで開きます。

```text
web_panel_opener/index.html
```

スマホではキャンバスが先に表示され、下の操作パネルを触りながらプレビューを確認できます。
操作パネルは折りたたみ式で、必要な時だけ開けます。
操作パネルは「モード」「基本操作」「背景」「文字・画像」「保存/読込」に分かれています。

## 設定ファイル

`設定をエクスポート` で、立ち絵/一枚絵・追加画像・文字・境界線・開いたパネル状態を含む `.json` ファイルを保存できます。
`設定をインポート` で、保存した `.json` ファイルを読み込むと状態を復元できます。

画像も `.json` 内に含めるため、ファイルサイズは画像の数や大きさに応じて大きくなります。

作業状態はブラウザ内にも自動保存されます。スマホブラウザがページを再読み込みした場合、起動時に前回状態を復元できます。

## 機能

- スマホのファイル選択から立ち絵/一枚絵画像を読み込み
- Canvas 上でシルエット化
- 背景色の変更
- 背景画像の読み込み
- シルエット色を共通パレットから選択
- 境界線をタップ操作で作成
- 境界線の色を共通パレットから選択
- 境界線で分割されたキャラクター領域をタップして開放
- 境界線の表示/非表示切り替え
- シルエット上に任意の文字・画像を複数重ねて表示
- 追加した文字・画像を背景上にも表示
- 追加した文字・画像をキャンバス全体へドラッグして位置調整
- 追加した文字を選択したまま内容・色・サイズを編集
- 追加した文字のフォントを変更
- 追加した文字をダブルクリック/ダブルタップしてキャンバス上で直接編集
- シルエット外側をタップした境界線の始点・終点を最寄りのシルエット境界へ吸着
- 立ち絵/一枚絵・追加画像・文字・境界線・開いた状態をJSONでエクスポート/インポート
- 作業状態の自動保存と起動時の復元
- スマホの長押しで保存用画像プレビューを表示

## 公開する場合

`web_panel_opener` フォルダを GitHub Pages などにアップロードすると、スマホから URL で利用できます。
