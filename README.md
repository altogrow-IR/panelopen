# シルエットパネル開け Web版

iPhone や Android のブラウザで使える静的Webアプリ版です。

## 使い方

`index.html` をブラウザで開きます。

```text
web_panel_opener/index.html
```

## 機能

- スマホのファイル選択から立ち絵画像を読み込み
- Canvas 上でシルエット化
- 境界線をタップ操作で作成
- 境界線で分割されたキャラクター領域をタップして開放
- 境界線の表示/非表示切り替え

## 公開する場合

`web_panel_opener` フォルダを GitHub Pages などにアップロードすると、スマホから URL で利用できます。
